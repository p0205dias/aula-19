Como Fazer Panqueca

Ingredientes

  • Ovos
  • Trigo
  • Leite
  • Açúcar
  • Fermento (Pó Royal)
  • Outros ingredientes opcionais (por exemplo, essência de baunilha ou sal)

Como Preparar Uma Porção:

  1. Para cada 1 ovo, adicione 200 ml de leite;
  2. Para 200ml de lite, 250g de trigo.
  3. para 250g de trigo, 100g da açucar
  4. para 100g de açucar, 15g de fermento
